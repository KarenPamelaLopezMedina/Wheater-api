# Wheater-api
Proyecto para mandar notificaciones del clima y recibir con sms.
# Objetivo:  Envío del pronósitco del tiempo al celular de manera diaria.
¿Qué se necesita?
- Una instancia de EC2
- Un script de python
- Api del clima

Procesaremos estos datos y compraremos un número de celular en Twilio,
el cual permitirá envíar mensajes desde tu número Twilio hasta tu celular de manera diaria.
